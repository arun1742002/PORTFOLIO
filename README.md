# Portfolio Project

This is a high-profile portfolio project showcasing various works and skills. The project is designed with a minimal and neat aesthetic, focusing on user experience and responsive design.

## Project Structure

```
portfolio
├── index.html          # Main HTML document for the portfolio
├── css
│   ├── style.css      # Main styles for the portfolio
│   ├── reset.css      # CSS reset rules for consistent styling
│   └── variables.css   # CSS variables for colors and fonts
├── js
│   └── main.js        # JavaScript for interactive features
├── assets
│   └── fonts          # Custom fonts used in the portfolio
└── README.md          # Documentation for the project
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the portfolio.
3. Customize the content in `index.html`, `css/style.css`, and `js/main.js` as needed.

## Features

- Responsive design that adapts to various screen sizes.
- Smooth scrolling for a better user experience.
- Mobile menu toggle for easy navigation on smaller devices.
- Clean and modern typography using custom fonts.
